# Contributing

This repository is for VybeCode binary releases and issue reporting only.

VybeCode is proprietary software owned by PrismWorks AI. The application source code is not published here, and this repository does not accept source-code contributions.

Official VybeCode desktop releases are free for unlimited personal, non-commercial use. Business, commercial, organizational, institutional, government, and enterprise use requires an appropriate written license from PrismWorks AI. Contact sales@prismworks.ai for business or enterprise licensing.

## Accepted Contributions

You may submit:

- bug reports
- install or update issue reports
- model or CLI agent issue reports
- documentation corrections for this public release-support repository

## Not Accepted

Do not submit:

- VybeCode source code
- reverse-engineered code
- proprietary project files
- API keys, tokens, secrets, or credentials
- confidential logs or screenshots
- third-party copyrighted material that you do not have rights to share

## Issue Quality

Good reports include:

- VybeCode version
- operating system and version
- exact steps to reproduce
- expected behavior
- actual behavior
- screenshots or logs with secrets removed
