# Support

For VybeCode desktop app problems, open a GitHub Issue in this repository:

https://github.com/prismworks-ai/vybecode/issues/new/choose

This repository is for release support and issue tracking only. It does not accept source-code contributions.

When reporting a problem, include:

- VybeCode version
- operating system and version
- whether the app was installed fresh or updated
- what you expected to happen
- what actually happened
- screenshots or logs with secrets removed
